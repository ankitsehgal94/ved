
# React Native vedantu mobile Application

# Getting started

1. Install React Native as described at https://facebook.github.io/react-native/docs/getting-started.html#content

2. Clone this repository

3. Run yarn , all required package dependencies will be installed automatically

# iOS

1. Run pod install from vedantuAssigment/ios folder
2. Run react-native run-ios from vedantuAssigment

# android

1. Run react-native run-android from vedantuAssigment
